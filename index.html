Welcome Survivor!

7 Days to Die is a Survival Horde Crafting game for various consoles and is available on Steam. The game is created by The Fun Pimps and was originally release in December of 2013. The intention of this website is to create a repository of information concerning this game, though please understand this is a work in progress. At the moment this project is in its infancy.
